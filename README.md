**О проекте**

Сайт представляет собой интерактивную страницу, где можно добавлять фотографии и удалять их.

**Функциональность**

* HTML
* Семантические теги
* CSS
* Flexbox
* Grid-layout
* Позиционирование блоков и элементов
* Адаптивная верстка под размеры экранов 1280px, 320px
* Псевдоклассы CSS
* Javascript
